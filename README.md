# Welcome to SBFR!

A web app for betting and having fun with the boys.

Build with ❤️ 
Uses Next.js, Vercel, and Firebase.

## Running Locally

```bash
npm install
npm run dev
```

## Building

```bash
npm install
npm run build
```

## Deployment 

```bash
sh ./push-env-to-vercel.sh
npm run deploy
```
