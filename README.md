# Welcome to SBFR!

A web app for betting and having fun with the boys.

Build with ❤️ using Remix Router and Firebase.

## Running Locally

```bash
npm install
npm run dev
```

## Building

```bash
npm install
npm run build
```

## Deployment 

```bash
firebase login
firebase init
npm run deploy
```
